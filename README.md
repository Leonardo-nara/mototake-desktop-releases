# Mototake Desktop Releases

Repositorio publico exclusivo para artefatos oficiais de distribuicao do Mototake Desktop.

Este repositorio pode conter apenas:

- instaladores `.exe`;
- arquivos `.blockmap`;
- `latest.yml`;
- notas de versao;
- checksums;
- releases e tags.

Este repositorio nao contem:

- codigo-fonte;
- credenciais;
- tokens;
- senhas;
- certificados;
- chaves privadas;
- dados de clientes;
- arquivos de banco de dados;
- logs sensiveis.
